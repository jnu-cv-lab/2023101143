## 项目简介

本项目是一个基于 Python 的简单图像处理实验，使用 OpenCV、Matplotlib 和 NumPy 对图像进行读取、显示、灰度化、裁剪和保存等操作。

该项目适用于初学者学习图像处理的基本流程。

## 项目功能

本项目实现了以下功能：

- 读取图片 `test1.jpg`
- 输出图像尺寸（高、宽、通道）和数据类型
- 显示原始图像（BGR 转 RGB）
- 将图像转换为灰度图并显示
- 保存灰度图为 `gray.jpg`
- 裁剪图像左上角区域并保存为 `crop.jpg`
- 输出图像左上角像素值
